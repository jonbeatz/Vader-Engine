BRAND IDENTITY & AESTHETIC
Design Philosophy
Dark-mode-first, always. The entire interface lives in the dark. Light mode is an afterthought (or omitted entirely).
"Forged in the dark" — industrial, raw, but polished. Not neon-overload cyberpunk; more like a refined black-ops developer terminal.
High information density when needed, but with clear visual hierarchy and breathing room.
Motion is purposeful — subtle animations for state changes, not decoration.
Color Palette
Primary Background: #0a0a0f to #0f0f1a (deep space black with subtle blue undertone)
Surface Elevations: #14141f, #1a1a2e, #1f1f3a (layered cards/panels)
Accent Color: #6366f1 (indigo) or #8b5cf6 (violet) — singular, consistent accent
Text Primary: #f8fafc (near-white)
Text Secondary: #94a3b8 (slate-400)
Text Muted: #475569 (slate-600)
Success: #10b981 (emerald)
Warning: #f59e0b (amber)
Error: #ef4444 (red)
Border/Divider: rgba(255,255,255,0.06) — nearly invisible, structural only
Typography
Headings: Inter or Geist (variable font), weight 600-700, tight tracking
Body: Inter or Geist, weight 400, line-height 1.6
Monospace (code/terminal): JetBrains Mono or Geist Mono, weight 400
Scale: Use a refined type scale — text-xs for metadata, text-sm for UI labels, text-base for body, text-lg for subheadings, text-2xl/3xl for page titles
Visual Language
Subtle glassmorphism on floating elements (modals, dropdowns, command palette) — backdrop-blur-xl with bg-white/5 and border-white/10
Glow effects on active/focused states — soft box-shadows in accent color at low opacity
Corner radius: rounded-lg (8px) for cards, rounded-md (6px) for buttons, rounded-full for avatars/pills
Shadows: Minimal, deep. shadow-lg with shadow-black/50 for elevated cards
Borders: 1px, low opacity, used for structure not decoration

INTERACTION & MOTION DESIGN
Animation Principles
Purposeful: Every animation guides attention or confirms an action
Fast: 150-300ms durations. No slow fades.
Easing: cubic-bezier(0.4, 0, 0.2, 1) for standard transitions
Specific Animations
Page transitions: Subtle fade + slight Y translate (8px)
Sidebar collapse: Width transition 200ms, content fades
Card hover: translateY(-2px) + shadow increase + border glow
Button press: scale(0.98) on click
Modal open: Backdrop fade 150ms, content scale(0.95→1) + fade
Toast enter: Slide from right + fade, 200ms
Chart data: Draw animation on mount, 800ms ease-out
Terminal/streaming text: Typewriter effect for AI output, 20ms per char

Scroll Behavior
Smooth scroll: Native CSS scroll-behavior: smooth
Sticky headers: Section headers stick when scrolling through long lists
Infinite scroll: For activity feeds, with loading spinner at bottom
