# VaderLabz

Dev Lab & AI Playground — Personal developer portfolio and experiment ground.

## Stack
- Next.js 14
- TypeScript
- CSS Modules
- Vader Protocol dark aesthetic

## Deploy to Hostinger
- Framework: Next.js
- Build command: `npm run build`
- Start command: `npm run start`
- Output directory: `.next`
- Node version: 20.x

## Local Dev
```bash
npm install
npm run dev
```
